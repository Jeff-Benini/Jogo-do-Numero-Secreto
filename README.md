<h2 id="sobre">Sobre</h2>
<h3>Esse foi um projeto no curso Desenvolvedor Full Stack do <a href="https://www.devclub.com.br/](https://www.alura.com.br/?srsltid=AfmBOoo8ZUBREMeOFG9WX7ZOKHIGQQZ4LVW3x4EEKIBou__5UmdGMzXS)">Alura</a> com foco em JavaScript visando criar um jogo que gera um número aleatório de um a mil e contabiliza os chutes até que o usuário acerte.</h3>

<h2 id="uso">Como Usar</h2>
<h2>
<ol>
  <li>Escolha e digite um número de um a mil.</li>
  <li>Clique no botão "chutar".</li>
  <li>Caso você acerte, o jogo irá retornar uma mensagem indicando o número de tentativas que você precisou para acertar.</li>
  <li>Caso você erre é só repetir o processo até acertar.</li>
  <li>Para iniciar um novo jogo é só clicar no botão "Novo Jogo".</li>
</ol>
</h2>
